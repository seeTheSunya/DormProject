# 🏠 도란(Doran) - 기숙사생을 위한 커뮤니티

도란(Doran)은 기숙사 생활을 더욱 즐겁고 편리하게 만들기 위한 기숙사생 전용 커뮤니티 플랫폼입니다.  
공동구매를 통해 배송비를 절약하고, 솔직한 사용 후기를 공유하며, 기숙사 생활 꿀팁과 고민을 나눌 수 있는 따뜻한 소통 공간을 지향합니다.

---

## 📅 프로젝트 개요

- **프로젝트명:** 도란 (Doran) - 기숙사생 공구/꿀팁/후기 게시판   
- **팀원:**  
  - 정준서 
  - 김보혜   
  - 이가희   

---

## 🛠️ 기술 스택 (Tech Stack)

**Backend**  
- Java 21, Spring Boot 3.2.3  
- JPA (Hibernate), MySQL 8.0  
- Spring AI (Google Gemini 연동)  

**Frontend**  
- HTML5, CSS3 (Tailwind CSS)  
- Vue.js 3 (CDN 방식)  
- Axios (API 통신)  

**Tools & API**  
- Google Gemini API (AI 글쓰기 보조)  
- Naver Shopping API (최저가 검색)  
- Git & GitHub (버전 관리)  
- DBeaver (DB 관리)

## 🎨 UI 및 주요 기능 (Features & Scenarios)

### 1. 메인 대시보드 (Main Dashboard)

**UI 구성**
- 상단 헤더: 로고, 테마 변경(라이트/다크/민트/핑크), 마이페이지, 로그아웃 버튼  
- 인기글 섹션: 최근 7일간 '좋아요' TOP 3 게시글 카드형 노출  
- 게시판 목록: 공동구매, 사용 후기, 레시피, 꿀팁, 고민상담소 등 카테고리별 최신글 5개 미리보기 제공  

**시나리오**
1. 사용자가 로그인 후 메인 페이지 접속  
2. 인기글 섹션에서 현재 가장 핫한 이슈 확인  
3. 원하는 게시판의 + 버튼 클릭 → 글쓰기 페이지 이동  
4. 제목 클릭 → 상세 페이지 이동  
5. 테마 선택 박스에서 '다크 모드' 선택 시 즉시 화면 변경  

---

### 2. 회원가입 및 로그인 (Auth)

**UI 구성**
- 로그인: 아이디/비밀번호 입력, 회원가입/아이디찾기 링크 제공  
- 회원가입: 아이디 중복 확인, 비밀번호 일치 확인, 이메일 입력  

**시나리오**
1. 비로그인 상태 → 로그인 필요 화면 표시  
2. 회원가입 시 아이디 중복 확인 후 이메일 입력  
3. 이메일 인증 링크 클릭 후 최종 가입 승인 및 로그인 가능  

---

### 3. 스마트 글쓰기 (Smart Editor)

**UI 구성**
- Quill Editor: 텍스트 서식, 이미지 삽입 가능  
- AI 도우미: "AI 글쓰기" 버튼 클릭 시 게시판 주제에 맞는 초안 자동 생성  
- 최저가 검색: "최저가 검색" 버튼 클릭 시 네이버 쇼핑 결과 검색 및 본문 삽입  

**시나리오 (공동구매)**
1. 사용자가 공동구매 게시판에서 글쓰기 클릭  
2. AI 입력창에 상품명 입력 후 [✨ AI 모집글 생성] 버튼 클릭  
3. AI가 초안 작성  
4. [🛍️ 최저가 검색] 클릭 → 상품 사진/가격 본문 삽입  
5. 모집 인원 및 1인당 가격 입력 후 등록  

---

### 4. 게시글 상세 및 소통 (Detail & Comments)

**UI 구성**
- 본문: 작성자, 작성일, 조회수, 내용 표시  
- 좋아요: 하트 버튼 (실시간 숫자 반영)  
- 댓글/대댓글 작성 가능  

**시나리오**
1. 게시글 확인 후 하트(❤️) 클릭  
2. 댓글 입력창에 궁금한 점 작성 또는 답글 작성  

---

### 5. 마이페이지 (My Page)

**UI 구성**
- 내 정보 확인, 비밀번호 변경, 회원 탈퇴 기능 제공  

**시나리오**
1. 비밀번호 변경: 현재 비밀번호 확인 후 새 비밀번호 설정  
2. 회원 탈퇴: 계정 삭제
